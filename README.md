# React 19 — State as a Snapshot Playground

A Vite + React 19 app to explore how state behaves like a snapshot per render, plus extra demos.

## Demos
- **Basics** — multiple updates in one render, async alert snapshot, async form snapshot.
- **Functional Updater** — contrast value form vs functional form of `setState`.
- **Async Race** — stale closure vs functional updates with setInterval.
- **Controlled vs Uncontrolled** — DOM-managed input vs React state.

## Quickstart
```bash
npm i
npm run dev
# open the URL Vite prints (e.g., http://localhost:5173)
```

## Build
```bash
npm run build
npm run preview
```
